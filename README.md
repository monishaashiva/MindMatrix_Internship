## 🧠 Side Project

### FocusNest – Productivity & Focus App

FocusNest is a productivity application designed to help students stay focused and manage their study sessions effectively.

Features implemented:

* Task management system
* Pomodoro-style focus timer
* Productivity statistics dashboard
* Daily streak tracking
* Clean Jetpack Compose UI

Tech Stack:

* Kotlin
* Jetpack Compose
* Android Studio
